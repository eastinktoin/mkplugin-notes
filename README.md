# mkplugin-notes

一个用于归档和发布独立 HTML 页面的仓库，存放各类笔记、工具页面与临时演示内容。

## 项目简介

本仓库主要用于保存和发布以 HTML 格式编写的独立页面。这些页面可能包含笔记、工具界面、演示原型或其他静态内容。每个页面均为自包含文件，不依赖外部服务或后端环境，可直接在浏览器中打开使用。

仓库不针对任何特定网站或域名，所有页面均为通用性内容归档。

## 目录结构

```
.
├── index.html          # 仓库主页或导航页（可选）
├── page1.html          # 示例页面1
├── page2.html          # 示例页面2
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── archive/            # 已归档的旧版本或不再维护的页面
    └── old-page.html
```

- 根目录下存放当前活跃或推荐的页面。
- `assets/` 目录存放页面引用的公共资源文件。
- `archive/` 目录用于存放已归档的旧版本页面，便于回溯。

## 页面归档说明

- 每个 HTML 页面应保持独立、自包含，尽量不依赖外部链接。
- 页面命名应简洁明了，反映内容用途。
- 如需引用公共资源，请统一放在 `assets/` 目录下。
- 归档的页面应移至 `archive/` 目录，并可在原位置留下重定向说明（可选）。

## 维护说明

- 本仓库由维护者不定期更新，添加新页面或整理归档。
- 欢迎提交 Issue 或 Pull Request 建议新增页面或改进现有内容。
- 所有页面均为静态内容，不包含用户数据收集或跟踪功能。
- 仓库不提供任何形式的担保或服务承诺，仅作为内容存档用途。
